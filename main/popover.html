Popover 的使用方法

点击查看弹出提示
名称 类型 默认值 描述
animation boolean true 在显示和隐藏提示框的时候是否使用动画过渡效果
html boolean false 在提示框内是是否以html形式注入数据。如果值为 false, 将会以 text 的形式将内容注入数据，使用text方式可以有效的防止XSS攻击。
placement string | function 'right' popover 的显示方位 - top | bottom | left | right
selector string false 如果提供了selector，将对符合条件的某个或多个元素启用工具提示。
trigger string 'click' popover 的触发方式 - click | hover | focus | manual
title string | function '' 当 `title` 属性未设定时所显示的默认值
content string | function '' 当 `data-content` 属性未设定时所显示的默认值
delay number | object 0

延迟显示和隐藏 popover (ms) - 不支持以manual为触发方式的 popover

如果只设定一个数值, 延迟将同时定义给 show/hide

用对象的定义方法如: delay: { show: 500, hide: 100 }
container string | false false

将 popover 添加到一个特定的容器中，定义方法如： container: 'body'
